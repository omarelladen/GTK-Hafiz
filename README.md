## Requirements (already pre-installed in most Linux distributions):
- GTK+3
- Python 3
- gobject-introspection

## Run the App:
```bash
python3 main.py
```

## Reset the Database:
### Install the sqlite3 CLI:
#### Debian:
```bash
sudo apt install sqlite3
```
### Run the Bash Script:
```bash
bash setup_db.sh
```
